Gráficas: Figurativas y no figurativas

Laura Rivera

En la creación de infografías se pueden utilizar gráficas figurativas y no figurativas, cada una cumple un propósito distinto.

GRÁFICA FIGURATIVA

La gráfica figurativa se utiliza para representar de una manera ilustrativa elementos relevantes al tema

GRÁFICA MIXTA

La gráfica figurativa se puede usar en conjunto con la gráfica no figurativa, para complementar la información que se puede entregar con solo una de aquellas.

GRÁFICA NO FIGURATIVA

Generen una gráfica no figurativa probando algún data sampLa gráfica no figurativa permite representar visualmente datos que no son visibles en la realidad.

La gráfica de arriba se despliega en la página como si se tratara de una imagen rasterizada, con el elemento <img>. Otros elementos de HTML podrían usarse para un mejor despliegue del SVG. Reviseramos tales elementos algunas clases más adelante, pero ya les conviene revisar Using SVG | CSS Trick.
